# 🏠 XGBoost Regression - California Housing Dataset

A machine learning project that uses **XGBoost Regressor** to predict housing prices using the California Housing dataset. The project covers data preprocessing, model training, evaluation, and prediction using a powerful gradient boosting algorithm.

## 🚀 Features

- Data preprocessing and exploration
- XGBoost Regressor implementation
- Model training and evaluation
- Housing price prediction
- Feature importance analysis

## 🛠 Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- XGBoost

## ▶️ Run the Project

```bash
git clone https://github.com/abhinav29890/XGBoost_Regression_Californiadatset.git

cd XGBoost_Regression_Californiadatset

python main.py
```
